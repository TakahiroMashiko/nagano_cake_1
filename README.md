* Ruby on Rails(フレームワーク)

# 推薦環境
Ruby 2.5 以降 Rails 5.2 以降

## インストール
```
$ git clone https://github.com/TakahiroMashiko/nagano_cake
$ cd nagano_cake
$ bundle install
$ rails db:seed
$ rails s
```
